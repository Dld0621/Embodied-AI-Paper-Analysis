# 话题分类说明

> 本文解释七条主线与 21 个细分主题的划分依据，以及一篇跨主题论文该归到哪个目录。

## 划分原则

分类服务于两个目的：**检索**（按主题快速找到相关笔记）与**对照**（把同类方法放一起看异同）。为此，主题划分遵循"按解决的问题分，而非按用的工具分"——同一份扩散模型，用于动作生成就归入扩散策略，用于视频预测就归入视频生成。

具身智能的能力可拆成一条链路：**感知世界 → 决定动作 → 驱动身体 → 在仿真中验证 → 用世界模型想象未来**。七条主线正对应这条链路的各个环节，再加上贯穿始终的遥操作与硬件。

## 跨主题论文如何归档

一篇论文常横跨多个主题。归档规则是**归到它最核心贡献所在的那个主题**，其余主题在笔记的"主题标签"里列出，并在相关目录的索引中加交叉链接。

例：一篇用高斯泼溅做可微仿真来训练抓取策略的论文，核心贡献若在仿真方法，归 `15-simulation`；若在抓取策略，归 `07-grasping`。判断标准是"去掉这部分，论文还成不成立"。

## 扩展分类

当某个细分主题笔记积累到一定规模（建议 15 篇以上），可在其目录下再开子目录细分，并在本文表格中补一行。新增主题需在 `notes/` 建对应编号目录，并同步更新本文件与主 README 的分类表。
